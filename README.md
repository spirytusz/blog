
# SpirytusZ的博客源码

## 初始化

```shell
# install hexo
brew install hexo

# clone source code
git clone git@github.com:spirytusz/blog.git

# install packages
cd blog
npm install
```

## 本地调试

```shell
hexo clean # clean

hexo g # generate html documents

hexo s # launch local server
```

## 部署

```shell
hexo clean # clean

hexo d # deploy
```
